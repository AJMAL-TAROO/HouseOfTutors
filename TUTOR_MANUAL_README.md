## Converting to MS Word Format

The manual is provided in Markdown format for easy editing and version control. To convert it to Microsoft Word (.docx) format, use one of these methods:

### Method 1: Using Pandoc (Recommended)

If you have [Pandoc](https://pandoc.org/) installed:

```bash
pandoc TUTOR_USER_MANUAL.md -o TUTOR_USER_MANUAL.docx
```

This creates a properly formatted Word document with all headings, lists, and formatting preserved.

### Method 2: Online Converters

Use a free online Markdown to Word converter:
- [Markdown to Word Converter](https://www.browserling.com/tools/markdown-to-word)
- [CloudConvert](https://cloudconvert.com/md-to-docx)
- [AnyConv](https://anyconv.com/md-to-docx-converter/)

1. Upload the TUTOR_USER_MANUAL.md file
2. Convert to DOCX format
3. Download the resulting Word document

### Method 3: Manual Copy-Paste

1. Open TUTOR_USER_MANUAL.md in a text editor
2. Select all content (Ctrl+A / Cmd+A)
3. Copy (Ctrl+C / Cmd+C)
4. Open Microsoft Word
5. Paste (Ctrl+V / Cmd+V)
6. Apply formatting manually:
   - Title: 20pt Bold
   - Heading 1 (#): 16pt Bold
   - Heading 2 (##): 14pt Bold  
   - Heading 3 (###): 12pt Bold
   - Body text: 11pt Regular
   - Font: Calibri or Arial
   - Line spacing: 1.15
   - Margins: 1 inch all sides
